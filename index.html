InterMine Apps/C Examples

A listing of InterMine JavaScript apps built using grunt-apps-c.

Examples

  • choose-list in a FlyMine account
  • component-400 displays results of an id resolution (list upload) job
  • list-widgets shows example List Widgets for a public FlyMine list
  • pathways-displayer compares pathways data across a bunch of InterMines
  • publication-search looks up publications associated with BioEntities in MouseMine
  • multimine-search searches multiple mines for keywords
  • Gene Recommendation Engine(Work In Progress)
